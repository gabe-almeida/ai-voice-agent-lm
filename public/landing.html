Architecture Overview

Our Voice Agent system is built with modern technologies to provide scalable, real-time voice interactions:

  • Backend: Node.js with Express and TypeScript
  • AI Models: Google Gemini 2.0 Flash for conversations
  • Voice Synthesis: Multiple TTS providers (Gemini, OpenAI, ElevenLabs)
  • Real-time Communication: WebSockets for low-latency streaming
  • Telephony: Twilio integration for phone calls
  • Audio Processing: Web Audio API for client-side audio handling
